## MDX的一些使用

### 执行javascript

export const currentDate = new Date()

```mdx:原文
export const currentDate = new Date();

`执行结果`:  {currentDate.toLocaleString()}
```

`执行结果`：{currentDate.toLocaleString()}

### 使用组件

export function Hello({ name }) {
  return <span>Hello {name}</span>
}

```mdx:原文
import { Hello } from "./hello";

`执行结果`:  <Hello name="路飞"/>
```

`执行结果`：<Hello name="路飞"/>
